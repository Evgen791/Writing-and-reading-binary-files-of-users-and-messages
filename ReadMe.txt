  Данный код создает и записывает объекты класса User и Message в два двоичных файла "users.bin" и "messages.bin".
Затем производится чтение из файлов данных объектов и вывод их полей на экран.
  Класс User описывает пользователя с поле имя, логин и пароль. Класс Message описывает сообщение, отправителя и получателя сообщения.
Чтобы записать объекты в файл, используется класс ofstream с режимом ios::binary и флагом ios::out. 
Объекты записываются в файл через метод write(), который принимает указатель на буфер и его размер.
  Для чтения объектов из файла используется класс ifstream с таким же режимом. Объекты читаются из файла в буфер через метод read().
Код читает из файлов по одному объекту классов User и Message. Если файлы содержат больше объектов, то они будут проигнорированы.
